Install with:

    git clone git://github.com/c10b10/vim-repo.git
    cd vim-repo.vim
    cp -R * ~/.vim
    cd ~/.vim
    mkdir bundle
    ruby update_bundles

If you want to install the Command-T plugin

    cd bundle/Command-T
    rake make
    
Repeat 

    ruby update_bundles

and

    cd bundle/Command-T
    rake make

whenever you feel like updating
